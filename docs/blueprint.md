# **App Name**: PlantãoAI

## Core Features:

- Demanda via Interface Estruturada: Permitir a criação de chamados de TI através de um formulário estruturado por categorias.
- Demanda via Texto Livre: Permitir a criação de chamados de TI através de um campo de texto livre.
- Processamento via Gemini API: Processar automaticamente os dados brutos inseridos pelo usuário, utilizando a API do Gemini, para gerar título, descrição técnica e resolução padronizados, atuando like a tool to ensure completeness.
- Exportação para Help Desk: Otimizar a exportação final dos chamados para o Help Desk oficial da instituição.

## Style Guidelines:

- Primary color: Soft blue (#89B4FA) to represent reliability and IT services.
- Background color: Very light blue (#E5EDF7), a desaturated variant of the primary color.
- Accent color: Soft green (#90EE90), providing a contrasting and fresh accent.
- Body and headline font: 'Inter', a sans-serif for modern readability and a clean user interface.
- Use consistent and clear icons related to IT support and common issues.
- Prioritize a clean and efficient layout to allow shift workers quick and easy acces to important functions
- Use subtle transition animations and loading indicators.
